**SİPER**, gökyüzünün ötesindeki verileri anlamlandırmak, uzay teknolojilerinde yerli ve milli inovasyon çözümleri üretmek amacıyla 
İMEJE Takımı tarafından geliştirilmiş bir Anti-spoofing platformudur.

---

## 🌌 Proje Hakkında

48 saatlik yoğun bir maratonun ürünü olan **SİPER**, Spoofing/Jamming sorunsalına odaklanır. 
"Milli Uzay Hamlesi" vizyonuyla uyumlu olarak, GNSS Mendeley verileri kullanılarak Anomali tespiti yapan model hedeflenmiştir.

## 🚀 Başlangıç (Kurulum)

Projeyi yerel ortamınızda çalıştırmak için:

1.  **Depoyu klonlayın:**
    ```bash
    git clone [https://github.com/kullaniciadi/imeje.git](https://github.com/kullaniciadi/imeje.git)
    ```
2.  **Gerekli kütüphaneleri yükleyin:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Uygulamayı başlatın:**
    ```bash
    python main.py
    ```
